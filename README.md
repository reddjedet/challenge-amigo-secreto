# ¿CÓMO FUNCIONA?

> [!NOTE]
> En este apartado vas a encontrar toda la información necesaria simular sorteos entre una lista de amigos.  
> El código incluye código html, css y javascript, y forma parte del desafío o challenge-amigo-secreto brindado por Alura LATAM.  
> Este proyecto busca ser usable por la mayor cantidad de personas posible. En la parte superior hay un título con una imagen de una persona sonriente sosteniendo un regalo. Abajo, campos para ingresar nombres y botones para añadir y sortear. El fondo está compuesto de 2 colores: azul claro y blanco. 

<img width="1350" height="601" alt="Sorteo" src="https://github.com/user-attachments/assets/29256bc2-cf4a-4b27-b4b0-216060d329cb" />  
  
- Ingreso de datos (nombres):
  - Ingresa los nombres de tus amigos en el cuadro
  - Clickea en `Añadir`
    - Cuando hayas terminado, clickea en `Sortear amigo`


# Posibles problemas y soluciones
- El código devolverá un alert si el usuario intenta agregar un campo vacío, procura ingresar nombres por input para evitarlo.
- Lo mismo pasaría en caso de intentar iniciar el sorteo sin datos ingresados. En este caso el aviso se emitirá por console.log (visible presionando F12).
